# ⚽ LA MASIA ELITE

A cinematic football training platform inspired by elite academies and modern sports-tech UI experiences.

Built with React + Vite, featuring immersive dark-mode visuals, animated training modules, dynamic workout systems, and position-based football coaching experiences.

---

# 🔥 Features

## 🎬 Cinematic UI/UX
- Premium dark-mode interface
- Smooth animations and transitions
- Glassmorphism-inspired cards
- Netflix / Nike Football inspired visuals
- Responsive layout for all devices

---

## ⚽ Position-Based Training System

Dedicated workout modules for:

- 🥷 Attacker
- 🎯 Midfielder
- 🛡 Defender
- 🧤 Goalkeeper

Each role has:
- Custom background visuals
- Specific drills
- Unique training videos
- Position-focused workout phases

---

## 🧠 Dynamic Workout Experience

Includes:
- Warmup
- Technical
- Shooting
- Fitness
- Recovery

Users can switch phases dynamically with smooth UI transitions.

---

## 🎥 Embedded Football Training Videos

Integrated YouTube football drills for:
- Ball mastery
- Passing
- Shooting
- Defensive movement
- Goalkeeping reflexes
